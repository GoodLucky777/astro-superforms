## Installation

<Steps>

1. First, install the `astro-superforms` package:

    <Tabs>
      <TabItem label="npm">
        ```bash
        npm install astro-superforms
        ```
      </TabItem>
      <TabItem label="pnpm">
        ```bash
        pnpm add astro-superforms
        ```
      </TabItem>
      <TabItem label="yarn">
        ```bash
        yarn add astro-superforms
        ```
      </TabItem>
    </Tabs>

2. Most package managers will install associated peer dependencies as well. If you see a `Cannot find package 'sveltekit-superforms'` (or similar) 
warning when you start up Astro, you'll need to install `sveltekit-superforms`:

    <Tabs>
      <TabItem label="npm">
        ```shell
        npm install sveltekit-superforms
        ```
      </TabItem>
      <TabItem label="pnpm">
        ```shell
        pnpm add sveltekit-superforms
        ```
      </TabItem>
      <TabItem label="yarn">
        ```shell
        yarn add sveltekit-superforms
        ```
      </TabItem>
    </Tabs>

3. Then, apply the integration to your `astro.config.*` file using the `integrations` property:

    ```js ins="superforms()" ins={2} title="astro.config.mjs"
    import { defineConfig } from "astro/config";
    import superforms from "astro-superforms";

    export default defineConfig({
      // ...
      integrations: [superforms()],
    });
    ```

</Steps>
